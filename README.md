## 两种用法，两种用户

打开 Movavi Slideshow Maker，第一个问题就是选模式——

**幻灯片向导（Slideshow Wizard）**：把图片和音乐拖进去，选一个主题模板（家庭回忆、假日旅行、爱情故事、生日纪念……），软件自动匹配过渡动画和背景音乐，一键生成。适合"我就是想快速出片、不想动脑"的场景。

**手动模式（Manual Mode）**：完整的多轨道时间轴，支持调色、蒙版、画中画、慢动作、文字特效等操作，和正经剪辑软件没什么区别，只是更简洁一点。适合"我有点想法，想自己捣鼓"的用户。

两种模式可以在同一个项目里来回切换——先用向导生成草稿，再切到手动模式精修，相当方便。

## 核心功能一览

- **素材库够用**：内置 160+ 种视觉特效和滤镜、105+ 转场动画、100+ 标题花字和趣味贴纸，加上内置音频素材库，日常出片基本不需要额外找素材
- **快速筛选**：内置特效商店，支持按关键词搜索和预览效果，不用一个一个点开看
- **格式支持广**：导入支持几乎所有主流图片/视频/音频格式，导出支持 MP4、MOV、FLAC、AVI 等 180+ 格式，还可以直接上传到网盘或视频平台
- **双平台通吃**：Win / Mac 均支持，购买一个授权码，两个平台都能用
- **录音录像**：支持麦克风录音和屏幕录像，可以直接在项目里配解说词

## 它的弱点在哪？

说点实话——

Movavi Slideshow Maker 对「专业剪辑师」来说肯定不够用，它没有专业调色节点、没有高级音效处理、没有多机位剪辑。它定位就是「不懂剪辑的普通人快速出片」，这个定位它做得很好，但如果你已经是个熟练剪辑手，可能会觉得功能限制有点多。
